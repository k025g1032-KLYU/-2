# 評価課題1: ヘッドジャンプ
## Intro
タイミングを見計らって敵を踏みつけて倒すゲーム

## 操作
1.A,Dキーで左右移動します\
2.SPACEキーでジャンプする

## ルール
1.踏みつけて敵を倒す

## 実装
1.二種類の敵(黒い奴機動力がすごいよ)\
2.マップチップの当たり判定(マップ単位)\
2.マップチップの当たり判定(ピクセル単位)\
3.マップチップで背景を描画\
4.ジャンプ\
5.斜め移動\
6.スクロール\
7.シーン切り替え\
8.SEとBGM\
9.オリジナル画像/動画\
10.敵に当たったら-hp/死ぬ\
11.敵を踏んだら-hp/倒す

## Installation
1.Zipを展開しまう\
2.exeを使用します

## 提出內容
1.Resources\
2.NoviceResources\
3.ヘッドジャンプ.exe\
4.Readme.md\
5.ScreenShot

## 頑張ったところ
1.マップ判定\
2.黒い敵の機動能力\
3.慣性\
4.画面素材全部自分で描きます

## 追加オリジナル仕樣
作り込まれた環境の中で追跡戦やバトルをしているような感覚を作り出したい。スピードはあえて追いつけないようにして、プレーヤーが適した環境やタイミングを見つけて攻撃するようにしたい。
