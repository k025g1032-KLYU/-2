# 評価課題1:  輝く宇宙へ（STGクラス化）
## Intro
クラス化のSTGゲームです

## 操作
1.WASD or 十字キーで移動します\
2.SPACEキーで発射する\
3.LSHIFTでダッシュ

## ルール
1.弾が当たら+１point\
2.敵を逃げたら-１point\
3.ぶつけるなら０point(敵を逃げるを防ぐため)\
Win :40ポイントゲット\
Lose:-15ポイント

## 実装
1.Playerクラス化\
2.Enemyクラス化\
2.カプセル化とアクセッサ\
3.STG要件\
4.シーンのクラス（SceneManager）（加点要件）\
5.弾クラス化（包含inPlayer）（加点要件）\
6.オリジナル画像/動画（加点要件？）\
7.SEとBGM（加点要件？）

## Installation
1.Zipを展開しまう\
2.exeを使用します

## 提出內容
1.Resources\
2.NoviceResources\
3.輝く宇宙へ.exe\
4.Readme.md\
5.ScreenShot

## 頑張ったところ
1.クラス化\
2.SceneManager\
3.画面素材全部自分で描きます

## 追加オリジナル仕樣
一般的な弾幕回避とは異なり、本作では防御的なプレイスタイルを試み、プレイヤー機体の衝突を有利な行動として成立させることで、取れる手段の幅とプレイの可玩性を高めています。
