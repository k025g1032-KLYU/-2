# 評価課題1:  STGクラス化
## Intro
クラス化のSTGゲームです

## 操作
1.WASDキーで移動します\
2.SPACEキーで発射する

## ルール
1.弾が当たら+１point\
2.敵を逃げたら-１point\
3.ぶつけるなら０point(敵を逃げるを防ぐため)\
Win :25ポイントゲット\
Lose:-25ポイント

## 実装
1.Playerクラス化\
2.Enemyクラス化\
2.カプセル化とアクセッサ\
3.STG要件\
4.シーンのクラス（SceneManager））\
5.弾クラス化（包含inPlayer）\
6.オリジナル画像/動画\
7.SEとBGM

## Installation
1.Zipを展開しまう\
2.exeを使用します

## 提出內容
1.Resources\
2.NoviceResources\
3.ヘッドジャンプ.exe\
4.Readme.md\
5.ScreenShot

## 頑張ったところ
1.マップ判定\
2.黒い敵の機動能力\
3.慣性\
4.画面素材全部自分で描きます

## 追加オリジナル仕樣
作り込まれた環境の中で追跡戦やバトルをしているような感覚を作り出したい。スピードはあえて追いつけないようにして、プレーヤーが適した環境やタイミングを見つけて攻撃するようにしたい。
